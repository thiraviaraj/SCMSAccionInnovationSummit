SCMS

Simplified Customer Messaging System

Communicate it in a simplest way.

Get Integration Steps Contact Us

Invisible plugin that handles messaging system with email integration, which enables the user to swiftly communicate instantly along with "screen recordings" / "Video" while staying on the same screen.
Problem Statement: When a customer faces any issue or wants to appreciate certain functionality, it gets inconvenient to look up the right email id, draft them, and send the expected details (screenshots, screen recording, etc.). Also interpreting their messages without relevant visuals is also hard.

Drafting Issue Steps is Time Taking Stuff

Listing steps one by one is difficult.

Multilanguage Issue.

Confusion on whom to cc and draft mail.

Developer understanding the issue.

3rd Party security issue.

Quicker resolution

Happier Customer.

Quick and simple.

Integration Steps 
Add Below to index.html and inside all iframes
<script src="https://cdn.jsdelivr.net/gh/thiraviaraj/SCMS@master/index.js"></script>
<link href="https://cdn.jsdelivr.net/gh/thiraviaraj/SCMS@master/index.css" rel="stylesheet" crossorigin="anonymous">
	
Configuration
SCMS.init(
      {
      	shortCutKeys : {
      		screen: 's',
      		audio: 'a',
      		video: 'v',
      		end: 'e',
      		appreciate: 'z',
      		issue: 'x'
      	},
      	mailAppreciateConfig: {
      		to: 'userssupport@accionlabs.com',
      		cc: 'manager@accionlabs.com',
      		subject: 'We are Happy About this feature!',
      		body: 'Hi Team,\n\n(<>) This feature has saved lot of time, onbehave of our team. I would like to thank all \n\n\n-\n<>'
      	},
      	mailIssueLoggerConfig: {
      		to: 'userssupport@accionlabs.com',
      		cc: 'escalation@accionlabs.com',
      		subject: 'Can Someone Look into this ?',
      		body: 'Hi Team,\n\n(<>) This feature is broken, Can someone look into the video attached and fix the issue as soon as possible.\n\n\n-\n<>'
      	},
      	hasFullStory: true
      });
